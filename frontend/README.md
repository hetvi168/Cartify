# 🛒 Cartify Frontend

The frontend application of **Cartify**, a full-stack MERN e-commerce platform built using React.js.

This client-side application provides a modern shopping interface with product browsing, authentication, cart management, and order functionality.

---

## 🚀 Features

- User authentication
- Product listing and details
- Shopping cart management
- User profile section
- Order history
- Responsive UI
- API integration with backend services

---

## 🛠️ Tech Stack

- React.js
- Redux Toolkit
- Context API
- React Router
- Axios
- CSS / UI Libraries

---

## 📦 Installation

Install dependencies:

```bash
npm install
```

---

## ▶️ Run Development Server

Start the React application:

```bash
npm start
```

The application will run at:

```
http://localhost:3000
```

---

## 🏗️ Production Build

Create an optimized production build:

```bash
npm run build
```

The generated files will be available inside:

```
build/
```
